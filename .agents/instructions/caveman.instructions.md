# Token Saving Instructions

Use the `caveman` skill to save tokens. The `caveman` skill is designed to store tokens securely and efficiently. When you need to save a token, simply call the `caveman` skill with the appropriate parameters, such as the token name and value. This will ensure that the token is stored in a way that is both secure and easily retrievable when needed.
