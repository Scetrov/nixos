# Project Constitution

## Research Preservation

- Any non-trivial investigation must write durable findings to markdown before the task is considered complete.
- Use `README.md` for project-facing knowledge that helps a human operate, debug, or continue the work.
- Use `AGENTS.md` for agent/operator memory, including failed attempts, working assumptions, exact commands, and hardware-specific lessons.
- Preserve sources and evidence links when web or vendor documentation informed the change.
- Record negative findings when they reduce uncertainty or prevent repeating failed work.
